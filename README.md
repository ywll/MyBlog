# is_my_life

个人博客备份

---
##  常用命令
### 新建文章  
```
hexo new [name]
```
### 提交部署 
```
hexo clean && hexo g && hexo d
```
### 将本地文件备份到github
```
git add .
git commit -m 'update'
git push
```
